Things to be implemented:
* Organizations

* Users
	* organization_id
	* firstname
	* lastname
	* age
	* email
	* password
* Shifts
	* organization_id
	* place_id
	* time_from
	* time_to
	* user_id - if null the shift is available
* Places
	* organization_id
	* name
	* min_age - minimum age required for working at this place (eg. clerk, minimum age of 16 in Denmark)
